Format for entering Field and Usages


For each field, there can be multiple usages. So for each field and n number of usages, you submit it like this:
<field> : <usage1> , <usage2> , <usage3> (and so on)

You can submit any number of these field-usages 'pairings' at once, with each 'pairing' separated by semicolons, like so:
<field1> : <usage1> , <usage2> , <usage3> ; <field2> : <usage3> , <usage> , <usage4> (and so on)
